碧水潭_碧水潭即红花尔基河瀑布，位于月亮小镇柴河西37公里处、红花尔基河源头，海拔930米。形成落差近10米的瀑布，瀑布宽3—5米，构成瀑布的基岩属1.4亿年以前的侏罗纪安山岩和玄武岩。绰尔河水汹涌澎湃,一带银瀑飞泻而下，涌入潭中，潭水清澈而幽深，雾气升腾。水潭两侧，古木参天，瀑布下面河道平坦变宽，积水形成一直径约30米的宽阔水潭，游人来此如入世外桃源，极适宜在此涉溪野营。_3_true_true
碧水亭_碧水亭——青沐城里一个著名景点里的亭子。那里，四季如春，蓝天碧水，耀眼的阳光映衬着，似乎格外美好。因湖水的颜色是碧绿的，碧水亭这个名字由此而来。也不完全算是四季如春吧，冬至过后，湖面便结了冰，听不见昔日潺潺的流水声，水温也因结冰而更加冷了。本是树林阴翳，鸣声上下，这会儿披上了一层厚雪，一眼望去是白茫茫一片，空中飘散着雪花，宛如人间仙境。鸟儿不想赏什么雪景，早在冬至前就成群结队的飞走了，好像是这里的常客，很了解一样。鸣声已失，白雪皑皑，这就是青沐城的冬天。女孩对那里的记忆早已模糊，只是每到了冬季，那一段记忆会在她的脑海中若隐若现。那时她还没有妹妹，母亲跟她说要带她去一个地方，那儿很美很美。_3_false_true
朝日峰_缙云山有九峰，东西排列。朝日峰，为九峰之首，迎沐晨曦，先露秀姿‘香炉峰，于悬崖绝壁之中生一石柱，状似香炉，峰顶断垣残门，古寨依稀，遗下当年风云史迹。狮子峰，突冗嵯峨，形同卧狮，上筑“太虚台”，登临俯览，嘉陵碧流，市井街貌，林海云雾，尽收眼底；缙云山岩层为三迭纪须家河组厚层砂岩组成，从东到西形成了九个峻拔雄秀的山峰，依次称作朝日峰、香炉峰、狮子峰、聚云峰、猿啸峰、莲花峰、宝塔峰、玉尖峰 ...缙云山与嘉陵江小三峡、合川钓鱼城一并被定为国家级自然风景名胜区。_3_false_false
飞流瀑_飞流瀑景区，位于太姥山岳景区西南侧，面积25平方公里。九鲤溪下游，霞浦县境内的杨家溪，相传因北宋名将杨文广平定南蛮十八洞而得名，自古是出入闽省的必经之地。南宋状元王十朋任泉州知府曾宿此地饭溪驿站，并赋诗赞云：“门拥千峰翠，溪无一点尘，松风清入耳，山月白随人”。_4_false_false
观云台_观云台——（又名轿子石），位于万岭小桥3公里处。相传，清朝同治元年(1862年)，清军在万岭山下官兴场被翼王石达开战败，战将唐友耕重伤不能行走，坐着轿子到此，太平军紧追不舍，轿夫见山高难抬，就丢下轿子逃跑，轿子也就留在那里，变成了今天路中央的大石头──“轿子石”。相传朱元璋在云游四海之时，正值冬天来到蜀南竹海，在忘忧谷遇到忘忧老人，老人见其虽蓬头垢面，但难压其霸气十足。于是问其出处，听后便让其往下看，朱元璋在观云台之上俯瞰，见青龙、见麒麟、见凤凰、见百龟、四临朝拜之相。心中大悦又见雪压竹枝欲沾泥，便挥笔提诗“雪压竹枝低 低头欲着泥 一朝红日起 依旧与在齐”。置身观云台，远望脚下块块水田又若片片明镜，拼接在青山绿水间，座座山丘好似千万只青绿的山龟正待你沙场点兵。雨雾天晴，浓云生聚，云海翻滚，气势恢宏，好似让你置生仙境。_5_true_true
红叶亭_爱晚亭，位于湖南省长沙市岳麓山下清风峡中，始建于1792年，名字来源于杜牧的七言绝句《山行》中的“停车坐爱枫林晚”。爱晚亭与陶然亭、湖心亭、醉翁亭并称中国四大名亭，也是革命活动胜地，为湖南省文物保护单位。亭形为重檐八柱，琉璃碧瓦，亭角飞翘，自远处观之似凌空欲飞状。内为丹漆园柱，外檐四石柱为花岗岩，亭中彩绘藻井，东西两面亭棂悬以红底鎏金“爱晚亭”，是由当时的湖南大学校长李达专函请毛泽东所书手迹而制。_4_true_true
花卉园_重庆花卉园，重庆渝北区百里花卉长廊工程，地处渝北区黄金地段，毗邻重庆北部新城开发区，该项目沿重庆机场高速公路和渝邻高速公路两旁规划布局。长廊南北两端距重庆市主城区分别为3公里、50公里，距寸滩深水港2公里，重庆江北国际机场位于长廊的中心地段，210、212、319国道、渝黔高速公路、渝怀铁路贯穿全区，区位优势明显，水陆空交通便捷。长廊内气候温和，雨量充沛，无霜期长，土地肥沃，水电路通讯等基础设施条件较好，是中外客商发展花卉的理想之地。长廊内有市级重点工程——重庆市现代农业园区。项目所在地——渝北区是“重庆市山水园林城区”，“重庆市环境保护模范区”，具有悠久的花卉发展史，是川派花卉的发源地。_2_false_false
九曲桥_杭州九曲桥位于杭州孤山北面，惠州九曲桥位于西湖中间，直通西湖平湖中的点翠洲（平湖和点翠洲是惠州西湖的），洲上有孤屿亭，明代建为点翠亭。九曲桥九曲十八弯，且每个弯曲的角度大小不一，有大于90度直角的，也有小于90度直角的。九曲桥如今的桥面为花岗石板，每一弯曲处一块石板上均雕刻一朵季节性花朵，如正月水仙、二月杏花、三月桃花……直到十二月腊梅；并在九曲桥头尾的两块石板上各雕刻一朵荷花。在湖心亭茶楼门前的一段桥面，中间雕刻一朵荷花，四角则分别雕刻彩云。_4_true_false
狮子山_端坐于香港九龙塘及新界沙田的大围之间，狮子山，同样端坐于香港人的精神高地，端坐于一条共同血脉的巍峨原点。狮子山在，精神就在，香港就在。狮子山为艰苦打拼的香港精神代言。狮子山早在1.4亿年前已形成，现在狮子山的位置，从前是一片大熔岩，岩浆由西贡及九龙喷出，后来火山停止爆发，大熔岩在海底慢慢冷却成了花岗岩，地壳的变动令花岗岩突出地面，成了新土地。狮子山上那头俯伏山顶的“狮子”，是由大自然操刀，经过不断的风雨侵蚀，直至最近的数百万年才形成的。香港大部分山都由火山岩形成，狮子山则是花岗岩。花岗岩不及火山岩能抵受风化侵蚀，但狮子山却十分特别，经过千百年仍可清晰见到“狮子”的头、身及尾。“狮子”的外形至今仍清晰可见，“狮子”头面向九龙西边，狮身连尾巴完整地伏在山上，相信再过一个世纪这头“狮子”仍会一直坐镇山上。_5_false_false

仙云石_这里有大片不规则的大块石头铺满山坡，仿佛一个巨石阵，村里人形象地将之比喻为“蚂蚁窝”。这石头层层叠叠，数不胜数，大小不同，形状有异，却无论大小均是岩中有岩，石中有石，在大块的岩石中有无数五颜六色的碎石镶嵌其中，呈现出斑驳陆离的奇彩。浙江大学地质系教授竺国祥和省地质调查院地质遗迹中心主任许红根等专家教授多次对仔牛山进行了实地勘察后断言，该遗迹形成于大约1亿——1.2亿年间的中生代时期，属侏罗纪晚期的火山喷发遗存。_5_true_false
一线天_一线天，是我国名山奇峰中常有的石景，因两壁夹峙，缝隙所见蓝天如一线而得名；较为知名者有峨眉山、华山、黄山、江郎山、武夷山、九宫山、三清山等处的“一线天”美景；由于“一线天”景观奇特，已成为许多风景名胜的热门景点。我国名山奇峰中常有的石景——一线天，风景奇特。从地质学的角度来看，最常见的“一线天”大都出现在石灰岩地区，是一种特殊的侵蚀地貌；但也有“一线天”是由断层形成，与前者相比，其不同处在于它们两侧的岩石有上下左右的位移。_5_true_true
